# Gyo 예제 프로젝트

## 기능

이 폴더에는 `gyo` 프레임워크의 다양한 기능과 API 사용법을 보여주는 예제 프로젝트들이 포함될 예정입니다. 사용자들은 이 예제들을 통해 `gyo`를 실제로 어떻게 활용할 수 있는지 쉽게 이해할 수 있습니다.

## 앞으로 추가할 기능 (Todo List)

- [ ] **`hello-world`:** 가장 기본적인 `gyo` 앱 구조를 보여주는 간단한 예제
- [ ] **`todo-app`:** 기본적인 상태 관리 및 사용자 입력을 처리하는 To-Do 리스트 예제
- [ ] **`native-api-test`:** `runtime`을 통해 모바일 기기의 네이티브 기능(카메라, GPS, 파일 시스템 등)을 호출하는 방법을 보여주는 예제
- [ ] **`ui-gallery`:** `gyo`에서 사용할 수 있는 다양한 UI 컴포넌트들을 모아놓은 갤러리 예제
- [ ] **`performance-test`:** 복잡한 리스트나 애니메이션 등 성능 테스트를 위한 예제
