# Gyo Runtime

## 기능

이 폴더는 `gyo`의 핵심 기능 중 하나인 **네이티브 브릿지(Native Bridge)**를 구현하기 위한 공간입니다. `runtime` 코드는 웹뷰(WebView)에서 실행되는 웹 애플리케이션(React)과 네이티브(Android/iOS) 플랫폼 간의 상호 통신을 가능하게 합니다.

웹 애플리케이션은 이 `runtime`을 통해 기기의 고유 기능(예: 카메라, GPS, 푸시 알림, 파일 시스템)에 접근할 수 있습니다.

- [ ] **카메라 API:** 웹에서 네이티브 카메라를 제어하고 사진/동영상을 가져올 수 있는 API 모듈 추가
- [ ] **GPS API:** 위치 정보(위도, 경도)를 얻어올 수 있는 API 모듈 추가
